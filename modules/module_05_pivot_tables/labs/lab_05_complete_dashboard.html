Lab 5: Complete Dashboard Creation

Build Professional Financial Dashboard

Lab Overview

Build a professional financial dashboard with KPI cards, breakdown tables, and interactive charts.

Prerequisites

  • Completed Modules 2-4
  • Data Model with all measures created

Financial Dashboard

Exercise 1: Prepare Layout

1Create Dashboard Sheet

  1. Create new sheet named Financial_Dashboard
  2. Select columns A ‚Üí L, right-click ‚Üí Column Width = 12
  3. Select rows 1 ‚Üí 31, right-click ‚Üí Row Height = 24
  4. Select range A1:L31, fill with background color #ECF0F1

Prepare dashboard layout with proper dimensions

Exercise 2: Dashboard Title

1Create Title Shape

  1. Insert ‚Üí Shapes ‚Üí Rectangle
  2. Fill color: #FFFFFF
  3. Type text: "Financial Dashboard"
  4. Font color: #2C3E50

Create dashboard title shape

Exercise 3: KPI Cards

1Prepare Pivot Data

  1. Create new sheet Data for pivot data preparation
  2. Create pivot table with measures:
    • Total Assets
    • Total Liabilities
    • Total Equity
    • Current Ratio
  3. Go to PivotTable Analyze ‚Üí OLAP Tools ‚Üí Convert to Formulas

Create pivot table for KPI data

Convert pivot to formulas using OLAP Tools

2Create KPI Card Shapes

  1. Insert ‚Üí Shapes ‚Üí Rectangle
  2. Select shape, press F2, then click the value cell in sheet "Data"
  3. Insert ‚Üí Text Box, type the label (e.g., "Total Assets")
  4. Select both shapes ‚Üí Right-click ‚Üí Group
  5. Repeat for Total Liabilities, Total Equity, Current Ratio

Create KPI card shapes linked to pivot data

Important: For all shapes, right-click ‚Üí Size and Properties ‚Üí Properties ‚Üí "Don't move or size with cells"

Exercise 4: Breakdown Tables

1Asset Breakdown Table

  1. Create new pivot table
  2. Drag Level_3 from Account_Hierarchy to Rows
  3. Drag measures to Values:
    • Total Assets
    • Total Asset PM
    • Total Asset vs PM
    • Total Asset vs PM %
  4. Rename headers:
    • Total Assets ‚Üí Ending Balance
    • Total Asset PM ‚Üí Prior Period
    • Total Asset vs PM ‚Üí Delta
    • Total Asset vs PM % ‚Üí Delta (%)
  5. Fill header color: #FFFFFF
  6. Select Delta column values ‚Üí Home ‚Üí Conditional Formatting ‚Üí Data Bars ‚Üí Solid Fill ‚Üí Green Data Bar
  7. Select Delta (%) column values ‚Üí Home ‚Üí Conditional Formatting ‚Üí Icon Sets

Create Asset Breakdown pivot table

Apply conditional formatting to breakdown table

2Liabilities and Equity Breakdown Tables

Repeat the same steps for Liabilities Breakdown and Equity Breakdown tables.

Exercise 5: Assets Doughnut Chart

1Create Doughnut Chart

  1. In sheet Data, create new pivot table
  2. Drag Level_3 from Account_Hierarchy to Rows
  3. Drag Total Assets measure to Values
  4. PivotTable Analyze ‚Üí PivotChart ‚Üí Pie ‚Üí Doughnut
  5. Right-click chart ‚Üí Hide All Field Buttons on Chart
  6. Right-click chart ‚Üí Add Data Labels ‚Üí Add Data Callouts
  7. Format chart colors (Format Chart Area ‚Üí Fill & Line):
    • Piece 1: #2C3E50
    • Piece 2: #3E5871
    • Piece 3: #52718E
    • Piece 4: #6B89A5
    • Piece 5: #89A3BB
  8. Border: No line
  9. Move chart to sheet Financial_Dashboard

Create Assets Doughnut chart with custom colors

Exercise 6: Liabilities Bar Chart

1Create Bar Chart

  1. In sheet Data, create new pivot table
  2. Drag Level_3 from Account_Hierarchy to Rows
  3. Drag Total Liabilities measure to Values
  4. PivotTable Analyze ‚Üí PivotChart ‚Üí Bar ‚Üí Clustered Bar
  5. Right-click chart ‚Üí Hide All Field Buttons on Chart
  6. Right-click chart ‚Üí Add Data Labels ‚Üí Add Data Label
  7. Format bar color: #2C3E50
  8. Move chart to sheet Financial_Dashboard

Create Liabilities Bar chart

Exercise 7: Equity Column Chart

1Create Column Chart

  1. In sheet Data, create new pivot table
  2. Drag Period_Label from Time_Periods to Rows
  3. Drag Total Equity measure to Values
  4. PivotTable Analyze ‚Üí PivotChart ‚Üí Column ‚Üí Clustered Column
  5. Right-click chart ‚Üí Hide All Field Buttons on Chart
  6. Right-click chart ‚Üí Add Data Labels ‚Üí Add Data Label
  7. Format column color: #2C3E50
  8. Move chart to sheet Financial_Dashboard

Create Equity Column chart by period

Exercise 8: Add Slicer for Filtering

1Create Slicer

  1. Select any pivot table created earlier
  2. PivotTable Analyze ‚Üí Insert Slicer
  3. Select the field to filter by (e.g., Period_Label, Level_1)
  4. Click OK

Insert slicer for interactive filtering

2Connect Slicer to All Pivots

  1. Right-click the new Slicer
  2. Select Report Connections
  3. Check all pivot tables to connect them to this slicer
  4. Click OK
  5. Move slicer to sheet Financial_Dashboard

Connect slicer to all pivot tables via Report Connections
